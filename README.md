# Math Club, ISI Bangalore site

## Setup

- Install Hugo: https://gohugo.io/installation/
- Clone the repo
  ```bash
  git clone --recurse-submodules git@github.com:mathclubisib/mathclubisib.github.io
  ```
- Update submodules
  ```bash
  git submodule update --remote --merge
  ```
- Serve to localhost
  ```bash
  hugo server
  ```
